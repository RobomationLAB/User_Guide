# 블록
## 언어 정하기
음성 인식 언어를 설정합니다.  
언어를 설정하지 않아도 자동으로 설정됩니다.


![](https://github.com/user-attachments/assets/b58e4965-9024-4ef3-9e52-5dd0664c7af8)


### 드롭다운 옵션 및 입력값
| 이름 | 구분 | 설명 | 범위 / 종류 |  
| ------ | ------ | ------ | ------ |
| language | 드롭다운 옵션 | 인식 언어 | en-US(English), ko-KR(한국어) | 
### 자바스크립트 코드
```javascript
// 음성 인식 언어를 English로 설정하기
$('ASR*0:lang').d = 'en-US';

// 음성 인식 언어를 한국어로 설정하기
$('ASR*0:lang').d = 'ko-KR';
```

### 파이썬 코드
```python
# 음성 인식 언어를 English로 설정하기
__('ASR*0:lang').d = 'en-US'

# 음성 인식 언어를 한국어로 설정하기
__('ASR*0:lang').d = 'ko-KR'
```

## 음성 인식 시작하기 / 중지하기
음성 인식을 시작하거나 중지합니다.


![](https://github.com/user-attachments/assets/327ee7a4-40f0-44b9-b4be-ebdd4bb3faec)


### 드롭다운 옵션 및 입력값
| 이름 | 구분 | 설명 | 범위 / 종류 |  
| ------ | ------ | ------ | ------ |
| toggle | 드롭다운 옵션 | 음성 인식 여부 | 시작하기(1), 중지하기(0) |
### 자바스크립트 코드
```javascript
// 음성 인식 시작하기
$('ASR*0:listen').d = 1;

// 음성 인식 중지하기
$('ASR*0:listen').d = 0;
```

### 파이썬 코드
```python
# 음성 인식 시작하기
__('ASR*0:listen').d = 1

# 음성 인식 중지하기
__('ASR*0:listen').d = 0
```
